## Colours

| Name | Hex | Used for |
|---|---|---|
| Soft Blush |  `#ffe5ec` | Background |
| Pastel Pink |  `#ffc2d1` | Borders, form inputs |
| Cherry Blossom |  `#ffb3c6` | Card borders |
| Cotton Candy |  `#ff8fab` | Hover effects |
| Petal Rouge | `#db5375` | Header, footer, buttons, headings |



## Typography

### Fonts
- **Parisienne** — used for all headings 
- **Playfair Display** — used for body text, navigation, buttons

### Sizes
| Element | Size |
|---|---|
| h1 (header title) | 64px |
| h2 (page headings) | 48px |
| h3 (card headings) | 32px |
| Body text | 16px |
| Small text | 14px |
| Very small text | 12px |


## Spacing
| Element | Value |
|---|---|
| Page margin (left/right) | 120px |
| Page margin (top)| 60px |
| Gap between cards | 30px |
| Card padding | 30px 35px |
| Header padding | 60px |
| Footer padding | 30px 40px |

### Layout Techniques
- **Flexbox** — used for the header (logo + navigation)
- **CSS Grid** — used for the place cards and outfit cards (2 columns)
