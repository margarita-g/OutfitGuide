# Outfit Guide
Als mein Projekt habe ich einen Outfit-Guide Website erstellt, auf dem man Outfit-Ideen für verschiedene Anlässe finden kann — zum Beispiel für Theater, Kino oder die Arbeit. Zu jedem Outfit gibt es eine kurze Beschreibung, welche Kleidungsstücke dazugehören und warum sie gut zusammenpassen.
 
Die Webseite besteht aus mehreren Unterseiten: einer Startseite mit einer kurzen Einführung, einer Übersichtsseite mit allen Outfit-Ideen und einzelnen Detailseiten pro Outfit mit weiteren Informationen.
